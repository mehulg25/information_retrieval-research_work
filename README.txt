The code has been written in Python using the Jupyter Notebook.
It can be run on the Jupyter Notebook after installing the following packages:
- pandas
- numpy
- nltk
- gensim

Steps to run the Code:
1. Download the dataset and code files.
2. Open the code files in Jupyter Notebook with Python 3.7.0 or above.
3. Update the destination of the dataset in the code and run the file.
